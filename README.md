### Back-End
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository\
[TypeORM](https://typeorm.io/#/) ORM connect to database using Query Builder

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```
